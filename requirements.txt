# YOLO detection (Stage 3+, not needed for color-mask mode)
# Two backends — pick ONE based on your hardware:
#
#   Option A: onnxruntime (RECOMMENDED for RPi / embedded)
#     - No torch required (~50MB vs ~500MB)
#     - Faster install, lower memory footprint
#     - Requires exporting YOLOv8n to ONNX first (one-time, on desktop):
#         pip install ultralytics && yolo export model=yolov8n.pt format=onnx
#         (or: python -c "from ultralytics import YOLO; YOLO('yolov8n.pt').export(format='onnx')")
#     - Then copy yolov8n.onnx to the RPi
#
#   Option B: ultralytics (easier to get started on desktop, overkill for RPi)
#     - Full torch stack, ~500MB, slow to install on RPi
#     - Works out of the box without manual ONNX export

# For RPi (Option A — recommended):
