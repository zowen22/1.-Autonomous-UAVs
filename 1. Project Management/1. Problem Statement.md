## Problem

Builder wants to develop a series of increasingly complex autonomous UAVs — from a quad interceptor to a full fixed-wing loitering strike platform — using accessible hobby hardware, open-source flight control software, and onboard computer vision. The gap is bridging the familiar FPV/Betaflight quad ecosystem into autonomous fixed-wing platforms with MAVLink-driven companion computer integration.

## Target User

Zach — experienced FPV quad builder, aviation club member, Comic Con / prop-building club member. Comfortable with the Betaflight/FPV stack. No hard budget limit. Some builds have prop/aesthetic goals alongside functional ones.

## Goals

- Build and fly three progressively complex autonomous UAVs
- Develop a real CV targeting pipeline (color mask → YOLOv8) running on onboard compute
- Learn ArduPlane / Pixhawk ecosystem for fixed-wing autonomy
- Produce a prop-quality Hunter Killer airframe for club display
- All code testable in SITL before any hardware flies

## Core Features

**Anti-Air Quad:** Detect an airborne target, lock optically, fly into it (kamikaze interceptor)
**Recon Fixed Wing:** Autonomous waypoint nav, loiter, photo/video capture
**Hunter Killer Fixed Wing:** Launch → loiter → CV target acquisition → autonomous engagement; matches Black Ops HK aesthetic

## Constraints

- Intel 2020 MacBook Air (development/Xcode work on separate machine)
- RPi 5 prices elevated due to 2026 LPDDR4 RAM shortage (~$80–$120); Orange Pi 5 is current better value
- Club field access needed for Stage 2+ autonomous flights
- FAA/local regs apply to autonomous flights; Stage 4 may require authorization
- CV targets must be inert markers — never autonomously target people

## Open Questions

- Companion computer: RPi 5 4GB vs. Orange Pi 5 (currently leaning OPi 5 on cost)
- Airframe path: OTS Zohd Dart XL now, scratch build HK for final
- CV target definition for demos: color-coded foam markers → YOLO COCO classes
- Engagement mode: recoverable (pull-up) vs. expendable (impact)
- Live video downlink: analog FPV for range, RTSP WiFi for demos
