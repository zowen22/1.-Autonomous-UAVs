# Project Overview

*Generated from Problem Statement: 2026-06-16*

## Summary

Three-build autonomous UAV project progressing from a hand-launched quad interceptor to a full fixed-wing loitering strike platform. Primary focus is the **Hunter Killer fixed wing** — a foam/composite flying wing running ArduPlane on a Matek H743-Wing FC with a companion computer (RPi 5 or Orange Pi 5) executing a YOLOv8 CV targeting pipeline over MAVLink. The HK also doubles as a prop build for the builder's Comic Con / prop club.

## Scope

### In:
- Anti-Air Quad (quad interceptor with CV targeting)
- Hunter Killer Fixed Wing (4-stage autonomy: manual → waypoints → CV acquisition → engagement)
- Recon Fixed Wing (long-endurance ISR platform, shares HK avionics platform)
- Full software stack: cv_pipeline.py, SITL test tooling, preflight checks
- ArduPlane configuration, BOM, wiring reference
- Aesthetic HK prop build (paint scheme, 3D-printed details, LED effects)

### Out:
- Production/commercial deployment
- Swarm coordination
- Live encrypted telemetry beyond SiK radios
- Any targeting of people or occupied areas
