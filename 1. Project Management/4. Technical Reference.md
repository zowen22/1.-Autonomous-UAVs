# Technical Reference

*Last updated: 2026-06-16*

-----

## Tech Stack

| Layer | Technology | Notes |
|---|---|---|
| FC (Fixed Wing) | Matek H743-Wing V3 | STM32H743, native ArduPlane, multiple UARTs |
| Firmware | ArduPlane (ArduPilot) | Waypoints, GUIDED, LOITER, TECS |
| FC (Quad, future) | Betaflight TBD | Anti-Air Quad build |
| Companion Computer | Orange Pi 5 (preferred) / RPi 5 | OPi 5: built-in 8 TOPS NPU, ~$70; RPi 5 elevated due to RAM shortage |
| CV Framework | OpenCV + YOLOv8n ONNX Runtime | No PyTorch dep — ~500MB smaller, 15–25 FPS on OPi/RPi |
| RC Link | ExpressLRS EP2 2.4GHz | Low latency |
| Telemetry | SiK 915MHz 100mW pair | MAVLink to Mission Planner / QGC |
| Companion→FC | UART (GPIO) → H743 UART3 | MAVLink2 at 57600 baud |
| GPS | Matek M10Q-5883 | GPS + compass |
| Airspeed | Matek ASPD-7002 | Critical for TECS |
| Simulation | ArduPilot SITL | Test all autonomy before first flight |
| Mission Planning | Mission Planner / QGroundControl | GCS |

## ArduPlane Key Parameters

```
# Frame
FRAME_CLASS         = 2        # Fixed Wing
ELEVON_OUTPUT       = 4        # Elevon mixing (S2=left, S3=right)

# Airspeed
ARSPD_TYPE          = 1
ARSPD_FBW_MIN       = 12       # m/s stall + margin
ARSPD_FBW_MAX       = 22       # m/s max cruise
ARSPD_USE           = 1

# TECS
TECS_CLMB_MAX       = 5        # m/s
TECS_SINK_MIN       = 2
TECS_PITCH_MAX      = 15       # degrees
TECS_PITCH_MIN      = -15

# Navigation
WP_RADIUS           = 30       # m
LOITER_RADIUS       = 100      # m
CRUISE_SPEED        = 15       # m/s
RTL_ALTITUDE        = 100      # m AGL

# Companion computer UART3
SERIAL3_PROTOCOL    = 2        # MAVLink2
SERIAL3_BAUD        = 57       # 57600

# Geofence
FENCE_ENABLE        = 1
FENCE_TYPE          = 7
FENCE_ACTION        = 1        # RTL on breach
```

## Wiring Summary

```
LiPo XT60 → ESC → motor
LiPo → 5V BEC → OPi/RPi + FC servo rail
H743 S1 → ESC (motor throttle)
H743 S2/S3 → Left/Right elevon servos
H743 UART1 → ELRS EP2 Rx (RC link)
H743 UART2 → SiK 915MHz (telemetry to ground)
H743 UART3 → OPi/RPi GPIO UART (MAVLink2 companion)
H743 I2C  → Matek M10Q GPS+compass
H743 ADC  → Matek ASPD-7002 airspeed
```

## Stage Gates

| Stage | Milestone | Go Criteria |
|---|---|---|
| 1 | Manual + GPS Stabilization | Stable FBWA 5+ min, telemetry live, RC failsafe → RTL |
| 2 | Autonomous Waypoints | Full 5-WP mission without intervention, loiter within 20m |
| 3 | CV Target Acquisition | Aircraft steers toward target in GUIDED, instant RC override |
| 4 | Full Autonomous Mission | Launch → loiter → CV lock → terminal approach |

## Safety Rules

- SITL test every new autonomous behavior before real flight. No exceptions.
- Geofence always enabled Stage 2+
- CV targets = inert color markers or foam. Never autonomously target people.
- Stage 4: club field, observer, dedicated kill switch on RC
- Check FAA / local regs before autonomous flights

## Known Constraints / Gotchas

- `asyncio.ensure_future` deprecated — use `asyncio.create_task()` (already fixed in cv_pipeline.py)
- Set `CAP_PROP_BUFFERSIZE=1` on camera to minimize latency in real-time guidance loop
- Accelerometer cal: level wing chord 2–3° nose-up (cruise attitude, not flat table)
- Compass optional for fixed wing at speed — disable if causing heading issues (`COMPASS_USE = 0`)
- ONNX Runtime: use `onnxruntime-lite` not full `onnxruntime` on OPi/RPi — avoids ARM build issues
